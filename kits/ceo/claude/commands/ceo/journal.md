$ARGUMENTS

Run the **founder-journal** skill. 10-minute Friday ritual:

1. Ask for 3 specific wins (push back if they list 1 or 5).
2. Ask for 1 named failure (no euphemisms).
3. Extract 1 testable lesson from the failure.
4. Force 1 concrete recalibration for Monday.
5. Energy check (3 lines).
6. Save to `founder-log/weekly/YYYY-WW.md` (append to existing if the Monday priorities file exists).

Keep total output under one printed page.
