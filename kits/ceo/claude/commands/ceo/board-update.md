$ARGUMENTS

Run the **board-update** skill. Process:

1. Ask the 6 intake questions: MRR + delta, cash + burn + runway, top-of-funnel metric, biggest win, biggest challenge (no spin), one specific ask.
2. If the CFO has run `/cfo:investor-update` this month, consume its KPI table output from `founder-log/investor-updates/YYYY-MM.md` instead of asking from scratch.
3. Write the 6-section update: Headline, KPIs (with MoM Δ), Wins, Challenges, Asks, What's next.
4. Keep the whole document under one printed page. Short sentences.
5. Save to `founder-log/investor-updates/YYYY-MM.md`.
6. Output the email-ready version the founder can paste into Gmail.

Refuse to invent news if the month was quiet.
