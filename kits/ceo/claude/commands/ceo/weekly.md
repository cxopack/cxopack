$ARGUMENTS

Run the **weekly-priorities** skill. You are in Monday-ritual mode:

1. Read the most recent entry in `founder-log/weekly/` if one exists.
2. Ask for a 60-second brain dump of what's on the user's mind right now.
3. Categorize each item: MOVE / LEARN / MAINTAIN / DEFER / KILL.
4. Cut to exactly 3 priorities — refuse a 4th.
5. For each: outcome, leading indicator, kill criteria, first Monday action.
6. Save to `founder-log/weekly/YYYY-WW.md`.
7. End with one provocative question for the founder to answer before tomorrow.

Cap the interaction at 15 minutes. Be opinionated. Don't accept vague outcomes.
