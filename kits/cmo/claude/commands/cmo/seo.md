$ARGUMENTS

Run the **seo-page** skill. Steps:

1. Identify intent (informational / evaluative / navigational / transactional).
2. Check the top 5 SERP results — match shape or deliberately break from it.
3. Require first-hand data (screenshot, benchmark, customer quote). If none, stop.
4. Write H1 + H2s that answer the question fully + table + 3 internal links + FAQ (People-Also-Ask) + meta description (150-160 chars).

Refuse thin content (<300 words) on competitive keywords. Save to `content/seo/<slug>.md` and add to the cluster index.
