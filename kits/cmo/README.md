# CxOPack — CMO Kit

> **Your head of marketing — positioning, content, SEO, launch, social.**

## What's inside
| Skill | Does |
|---|---|
| **positioning** | April Dunford style: For [audience] who [struggle], [Product] is a [category] that [unique value]. Force disqualifiers. |
| **seo-page** | Programmatic SEO page generator with intent match and internal link plan |
| **landing-copy** | Landing page copy with verifiable claims only — no fluff |
| **linkedin-batch** | 7-post LinkedIn batch (one per weekday + 2 weekend), varied formats |
| **launch-comms** | Launch day: tweet thread, LinkedIn, newsletter, PR pitch |
| **brand-voice** | Voice doc: 5 rules, 5 anti-rules, 10 example before/after |
