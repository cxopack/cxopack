$ARGUMENTS

Run the **cold-outbound** skill. Require the ICP doc (`sales/icp.md`). For each prospect, pull 3 specifics (recent action, tech choice, public metric) before writing.

Format: 4 messages — D0 (≤60 words), D3 (≤50), D7 (≤35, soft breakup), D11 (≤30, hard breakup).

Bans: "hope this finds you well", "quick question", feature lists in msg 1. Message 1's ask is a yes/no question, never a meeting request.

Save to `sales/outbound/<segment>-YYYY-MM-DD.md`.
