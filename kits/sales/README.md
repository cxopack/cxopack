# CxOPack — Sales Kit

> **Your head of sales — pipeline, outbound, closing.**

ICP definition, cold outbound, discovery call script, objection handling, proposal generator, pipeline review.

## Install
Same pattern as other kits (`claude/`, `chatgpt/`, `cursor/`, `prompts/`).

## What's inside
| Skill | Does |
|---|---|
| **icp-workshop** | Interview-style ICP definition — output: 1-pager with disqualifiers |
| **cold-outbound** | 4-message sequences. Short, specific, deletable — not salesy. |
| **discovery-script** | 45-min call script: 70% listen, 30% frame. No pitching on discovery. |
| **objections** | Library of 20 common objections + proven responses |
| **proposal** | SOW / proposal generator tied to discovery notes |
| **pipeline-review** | Weekly 15-min review ritual |
